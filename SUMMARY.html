Architecture

Input Tokens (B, T) ↓ Token Embedding — nn.Embedding(2048, 160) // no positional embedding (ALiBi) ↓ Depth-Recurrent Blocks [0, 1, 2, 3, 2, 3] — 4 physical → 6 effective layers └ Each block: │ RMSNorm → SelfAttention (4 heads, ALiBi bias, Value Residuals) → residual add │ RMSNorm → SwiGLU MLP (E=3, dim 160→480→160, bias=False) → residual add ┐ ↓ RMSNorm (final) ↓ Output Head — nn.Linear(160, 2048) // weight-tied to Token Embedding ↓ Logits (B, T, 2048) → cross_entropy loss

Key Design Decisions

1. BPE Tokenizer (V=2048) Highest Impact

The corpus is ~14% Devanagari (train) / ~20% Devanagari (dev). A byte tokenizer encodes each Hindi character as 3 tokens, wasting 3× the context window on Hindi text. Our BPE achieves 2.51 bytes/token, giving each 256-token attention window an effective reach of ~642 bytes vs the baseline's 128 bytes (5× improvement).

2. Weight Tying Parameter Efficiency

Output projection shares weights with input embeddings, saving 327,680 parameters (2048 × 160). These saved parameters are reallocated to more attention/MLP capacity through the larger effective layer count.

3. ALiBi Positional Encoding Parameter Efficiency

Replaces learned positional embeddings with head-specific linear biases. Saves block_size × n_embd = 40,960 parameters and generalizes to positions unseen during training without any learnable parameters.

4. Depth Recurrence [0,1,2,3,2,3] Free Depth

4 physical transformer blocks are traversed in a 6-step recurrence pattern, revisiting blocks 2 and 3 twice. This doubles their training signal and gives the model 6 effective layers of depth at zero additional parameter cost.

5. Value Residuals (ResFormer) Deep Network Stability

Each attention layer gates its values with v = lam_v2 × v + lam_v1 × v1_cache, where v1_cache is the value tensor from the first layer. This prevents the attention collapse that typically occurs in deep transformer networks. Only 10 learnable scalar parameters total (2 per physical block).

6. SwiGLU MLP & RMSNorm Efficiency

SwiGLU (gate × up projection with SiLU activation) strictly outperforms GELU in compute-matched comparisons (LLaMA, PaLM). RMSNorm is ~15% cheaper than LayerNorm while being equally effective, as it skips the mean subtraction step.
